03 / ScrollTrigger 实战

滚动驱动的
动画叙事

向下滚动，体验 GSAP ScrollTrigger 的力量——固定、联动、视差，一切随你掌控。

SCENE 01

极少的代码

ScrollTrigger 让你用几行代码就能创建复杂的滚动动画。无需繁琐的事件监听，无需手动计算滚动位置。

gsap.to('.box', {
  scrollTrigger: {
    trigger: '.box',
    start: 'top center',
    scrub: true
  },
  rotation: 360,
  duration: 3
});
SCENE 02

固定 & 联动

将元素固定在视口中，同时驱动动画进度，创造沉浸式叙事体验。

SCENE 03

视差滚动

不同元素以不同速度移动，营造深度和空间感。每一层都有自己节奏。

多层视差 · 不同速度
SCENE 04

无限可能

从简单的淡入到复杂的时间线编排，ScrollTrigger 都能胜任。这就是行业标准的实力。

04 / 插件生态

强大的插件与缓动函数

可选插件和缓动函数让你轻松实现高级效果——滚动、变形、文本拆分、运动路径、FLIP 动画……应有尽有。

ScrollTrigger

滚动动画

行业标准级滚动驱动动画。创建令人惊叹的滚动效果——固定、联动、视差、进度条，用极少的代码实现。

了解更多

SplitText

文本拆分

将文本拆分为字符、单词或行，实现逐字逐行的精细动画。创建引人注目的标题动画和文字效果。

了解更多

MorphSVG

SVG 变形

在任意两个 SVG 路径之间实现平滑变形动画。从圆形到星形，从简单到复杂，变形从未如此优雅。

了解更多

MotionPathPlugin

运动路径

让元素沿任意 SVG 路径运动。创建曲线飞行、轨道旋转、波浪前进等复杂运动轨迹。

了解更多

Flip

FLIP 动画

First-Last-Invert-Play 技术。在任何状态变化之间创建平滑过渡动画——列表重排、布局变化、视图切换。

了解更多

Observer

事件标准化

在浏览器和设备间标准化事件检测。统一处理鼠标、触摸、滚轮、指针事件，简化跨平台交互开发。

无论你使用 CDN、NPM 还是模块打包器，GSAP 都能轻松集成到你的项目中。核心库包含所有实用方法和缓动函数。

感谢 Webflow，GSAP 现在完全免费，包括所有高级插件如 SplitText、MorphSVG 以及其他 Club GSAP 会员专属功能。即使是商业用途也完全免费！

1

CDN 引入

HTML 
<!-- GSAP 核心 -->
<script src="https://cdn.jsdelivr.net/npm/gsap@3.15/dist/gsap.min.js"></script>

<!-- ScrollTrigger 插件 -->
<script src="https://cdn.jsdelivr.net/npm/gsap@3.15/dist/ScrollTrigger.min.js"></script>
2

NPM 安装

Terminal 
npm install gsap
3

模块导入

JavaScript 
// 典型导入方式
import gsap from "gsap";
import ScrollTrigger from "gsap/ScrollTrigger";
import Flip from "gsap/Flip";
import Draggable from "gsap/Draggable";

// 别忘了注册插件
gsap.registerPlugin(ScrollTrigger, Draggable, Flip);
